# deva-fiber
```
curl --location --request GET 'localhost:8000/power/:column/:numRecord'
```
You can replace value column with all, active_power, power_input
and replace numRecord do you want max is 1000 records.
 ## Description
 to run project 
 step 1:
 ```
 make build
 ```

 step 2:
 ```
 make run
 ```

 to stop project
 ```
 make stop
 ```
